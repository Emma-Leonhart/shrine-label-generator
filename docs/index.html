Shrine Label QuickStatements

GitHub repository

This project generates multilingual Wikidata labels for Shinto shrines and Buddhist temples. Starting from Japanese names fetched via SPARQL, the pipeline produces labels in 12 languages using a combination of custom phonological rules, transliteration libraries, and character substitution.

Each page below contains the full QuickStatements batch for that language, ready to paste into QuickStatements and import into Wikidata.
